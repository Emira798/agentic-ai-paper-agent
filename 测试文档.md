## 🔍 故障排除

### 常见错误

#### API 密钥错误
```
Error code: 401 - Incorrect API key provided
```
**解决方案：**
- 检查 API 密钥是否正确设置
- 验证密钥是否有余额
- 检查网络连接

#### 超时错误
```
TimeoutError: Request timed out
```
**解决方案：**
- 增加超时时间：`export TEST_TIMEOUT=120`
- 检查网络连接
- 减少测试范围

#### 余额不足
```
Error: Insufficient balance
```
**解决方案：**
- 充值 API 账户
- 联系服务提供商
- 使用更简短的测试查询

#### 数据库错误
```
sqlite3.OperationalError: database is locked
```
**解决方案：**
- 确保没有其他进程使用数据库
- 重启测试
- 使用内存数据库

### 调试技巧

1. **逐步测试**：先运行基础测试，再运行 API 测试
2. **详细日志**：使用 `-v --tb=long` 查看详细错误信息
3. **隔离问题**：单独运行失败的测试用例
4. **环境检查**：验证环境变量和依赖
